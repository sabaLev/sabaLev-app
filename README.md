# מחשבון מערכת סולארית

Калькулятор для расчета материалов для солнечных систем

## Функции

- ✅ Выбор типов панелей
- ✅ Ввод групп панелей (вертикальные и горизонтальные)
- ✅ Автоматический расчет необходимых материалов
- ✅ Редактирование рассчитанных значений
- ✅ Добавление каналов и дополнительных частей
- ✅ Экспорт отчета в HTML/PDF
- ✅ Поддержка иврита (RTL) и русского языков
- ✅ Адаптивный дизайн для мобильных

## Технологии

- **React 18** - UI фреймворк
- **TypeScript** - Типизация
- **Vite** - Сборщик
- **Supabase** - База данных (опционально)
- **Zustand** - Управление состоянием
- **i18next** - Интернационализация

## Установка

```bash
# Установить зависимости
npm install

# Запустить dev сервер
npm run dev

# Собрать для production
npm run build

# Просмотреть production версию
npm run preview
```

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── calculator/      # Компоненты расчетов
│   ├── export/          # Компоненты экспорта
│   ├── groups/          # Компоненты групп
│   ├── layout/          # Общие компоненты layout
│   └── panels/          # Компоненты панелей
├── services/            # Бизнес-логика
│   └── calculationEngine.ts  # Расчеты
├── styles/              # CSS стили
├── types/               # TypeScript типы
└── App.tsx             # Главный компонент
```

## Использование

1. **Выберите тип панели** - Выберите из доступных опций
2. **Введите группы** - Укажите количество панелей и групп (вертикальные и горизонтальные)
3. **Рассчитайте** - Нажмите кнопку "חשב" (Рассчитать)
4. **Отредактируйте результаты** - При необходимости измените рассчитанные значения
5. **Экспортируйте** - Выберите формат экспорта (HTML, печать)

## Развертывание

### На Vercel (рекомендуется)

```bash
# Установить Vercel CLI
npm install -g vercel

# Развернуть
vercel
```

### На GitHub Pages

```bash
npm run build
# Загрузить содержимое dist/ на GitHub Pages
```
